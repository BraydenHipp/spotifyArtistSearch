# Spotify Web Api Project
This project was my first into into web development. This app allows you to search for any artist and see a concise view of their discography, which includes their profile picture and their 10 most recent albums. 

## Usage
If you wish to use this project for yourself you will need to go into the app.js file and in the fetch new token function, replace client id and client secret with your own credentials from the Spotify developer dashboard.
